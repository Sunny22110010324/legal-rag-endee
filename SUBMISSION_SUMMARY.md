# Submission Summary: Legal RAG Assistant

## Project Overview
A complete AI-powered legal document analysis system using:
- **Endee Vector Database** for semantic search
- **RAG Pipeline** for intelligent document retrieval
- **Flask Web Application** with modern interface

## How to Run
```bash
# 1. Clone repository
git clone [repository-url]

# 2. Install dependencies
pip install -r requirements.txt

# 3. Start application
python app.py

# 4. Open browser: http://localhost:5000
